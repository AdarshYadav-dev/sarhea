# Sarhea 

A responsive “About Us” section built using **React + Vite + Tailwind CSS**.  
It adapts seamlessly across mobile, tablet, and desktop screens.

## Tech Stack
- React
- Vite
- Tailwind CSS

## Features
- Fully responsive layout  
- Clean and modern design  
- Component-based structure  
- Ready to integrate into any website  
---

## Getting Started
```bash
git clone https://github.com/AdarshYadav-dev/sarhea.git
cd sarhea
npm install
npm run dev
